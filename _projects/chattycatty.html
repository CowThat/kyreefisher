This was a semester long project for my software development class that followed the steps of software development from beginning to end.

requirements

For the project, we had to figure out all of the functional and nonfunctional requirements of the program. We also figured out the process we would use in development.

prototypes

These high-fidelity wireframes were made very quickly for the project. Since the project was focused on software development instead of just design, I didn't put as much focus into making sure everything was perfect. Still, I created a quick brand and created mockups of what every page we needed would look like.

testing the code

Then, we figured out where our code would need to be tested and created the test cases. This was a long document that outlined every possible feature in the program that needed testing and how we would test it.

user flow
